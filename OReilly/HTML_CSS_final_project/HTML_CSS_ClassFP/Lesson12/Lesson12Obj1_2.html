Getting Away from It All

I love going out hiking for the day, though I rarely seem to have the time to do so any more. The following are some of my favorite places to go in Indiana. Some offer more opportunities for hiking than others. Some offer more chances to get away from the hustle and bustle of people than others, particularly during during the fall and winter.

  • Indiana Dunes State Park
  • Turkey Run State Park
  • McCormick Creek State Park
